# EMSCodeSim Vitals – Vitals & Assessment Trainer

EMSCodeSim Vitals is a mobile-friendly EMS education app designed for EMT/EMS students and instructors to practice core assessment skills.

Included training modules:
- Blood Pressure Simulator (manual cuff practice)
- Pulse Test / Pulse Trainer
- Stroke Assessment (BE-FAST)
- Pupil Assessment
- Rule of Nines Burn Assessment
- Breath Sound Simulator

Educational use only. Not medical advice.
